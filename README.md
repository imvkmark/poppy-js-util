# Poppy Util

## Usage

### Install

```shell
$ pnpm install poppy-util
```

使用

```
// es
import { isMobile } from 'poppy-util';
```

```
// browser
<script src="node_modules/poppy-util/dist/poppy-util.iife.js"></script>
```

### Publish

```
nrm use npm && pnpm publish --no-git-checks --access=public && nrm use taobao
```
